US Sales Analytics using Python

Project Overview
This project analyzes a detailed US sales dataset to identify trends in sales, profit, regions, products, and discount impact using Python.

Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

Key Analysis Performed
- State-wise and region-wise sales analysis
- Monthly sales trend analysis
- Product-level sales and profit comparison
- Discount vs profit relationship analysis

Key Insights
- Mobile products are the top contributors in both sales and profit.
- Certain regions outperform others in total revenue.
- Higher discounts increase the likelihood of negative profit.
- Seasonal patterns are visible in monthly sales trends.

Conclusion
This analysis helps understand business performance across geography, products, and pricing strategies, enabling better data-driven decision-making.
